# Netflix_2.0
Developed Netflix Clone built using MERN Stack (MongoDB, Express, ReactJS, NodeJS), Redux, TMDB API, Tailwind CSS & MUI.

Features:
- Login/Sign Up

- Browse (After Authentication)
    - Header.
    - Main Movie Container Portion:
        - Trailer in Background.
        - Title & Description.
    - Movie Container Portion:
        - MovieLists:
            - (Now Playing) * N.
            - (Popular Movies) * N.
            - (Top Rated Movies) * N.
            - (Upcoming Movies) * N.
        - Play Movie:
            - MovieDialog Popup
                - Video
                - Cancel    
    - Search Page:
        - Search Bar.
        - Movie Suggestions.
        - Play Movie:
            - MovieDialog Popup
                - Video
                - Cancel
